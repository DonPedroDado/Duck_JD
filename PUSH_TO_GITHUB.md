# Push Duck_JD to GitHub

1. Create the repo on GitHub: https://github.com/new?name=Duck_JD  
   - Name: **Duck_JD**  
   - Do **not** add README, .gitignore, or license.

2. In this folder, run (replace `YOUR_GITHUB_USERNAME` with your GitHub username):

```bash
git remote add origin https://github.com/YOUR_GITHUB_USERNAME/Duck_JD.git
git push -u origin main
```

If you use SSH:

```bash
git remote add origin git@github.com:YOUR_GITHUB_USERNAME/Duck_JD.git
git push -u origin main
```

3. After pushing, the source code will be at:  
   `https://github.com/YOUR_GITHUB_USERNAME/Duck_JD`
